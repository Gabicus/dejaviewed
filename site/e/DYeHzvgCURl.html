1k ft — Deep-dive / try it

Built a system that turns taste into data and data back into art. Started with a moodboard of references I’d been collecting — diagrams, memes, scans, glyphs, ritual geometry, fictional interfaces. Ran it through a process that extracts the meta from each piece: tags, signal, structure, palette. Combined the lot into a liminal atlas — a space organized by tags and by my personal philosophy. From there, an LLM steered by that philosophy generates brand new code-art interfaces. Live HTML. Live canvas. Each artifact thematically derivative of the original moodboard, each one novel.
