1k ft — Deep-dive / try it

Built a sort of “EMP grenade” using an ESP32, an nRF24 module, a battery board, and a simple web interface. The device creates heavy interference on the 2.4 GHz spectrum, which is the same band used by WiFi, Bluetooth, wireless speakers, headphones, smart devices, and a lot more. For this build I programmed the ESP32 to host its own WiFi control panel where you can arm the device and trigger a countdown timer before activation. After that, the RF module starts transmitting noise across multiple channels simultaneously. Everything fits into a tiny portable enclosure powered by a rechargeable battery.
