Let me know if you want this skill.. This is a 90-second walkthrough of how to turn your taste into a generative design system. The process is simple: Collect a moodboard. Extract the patterns. Tag the visual DNA. Write the philosophy behind it. Turn that into a structured atlas. Use an AI agent to generate new live interfaces from the rules. The point is not to copy reference images. The point is to extract the meta: color behavior, layout pressure, motion, density, texture, symbols, interface logic, emotional tone, and the worldview underneath the collection. Then you use that data to generate new artifacts that feel related to the source material without directly harvesting it. Taste -> data -> art.
