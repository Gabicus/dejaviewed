About DejaViewed
You’ve saved this before.

Every scroll session leaves a trail — posts you swore you’d come back to, tutorials half-remembered, repos lost in the feed. DejaViewed pulls everything out of your Saved tab and hands it to Claude: classified, tiered, cross-linked, and ready to use. What was once a graveyard of good intentions becomes a living catalog — searchable, explorable, and wired into your AI workflow.

Install

Clone Gabicus/dejaviewed — adds /dejaviewed to Claude Code.

git clone git@github.com:Gabicus/dejaviewed.git ~/.claude/plugins/dejaviewed Plugin install
git clone git@github.com:Gabicus/dejaviewed.git /tmp/dv && cp -r /tmp/dv/skills/dejaviewed ~/.claude/skills/ Skill-only
/dejaviewed Invoke

Prerequisites

pip install pyarrow requests browser-cookie3 playwright Python deps
pip install yt-dlp faster-whisper imageio-ffmpeg Transcription (optional)
cp -r ~/.config/google-chrome/Default .profile-copy/Default Chrome profile copy

Requires Python 3.10+, Chrome profile with active IG session, Playwright MCP server. See full setup guide.
